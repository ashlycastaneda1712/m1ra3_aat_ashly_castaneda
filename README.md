# Ejemplo m1ra3_aat_ashly_castaneda

## Descripción General
Este sitio web tiene como objetivo simular un portal institucional que permite gestionar información de empleados y realizar cálculos de nómina complejos. El proyecto pone énfasis en la maquetación precisa (fiel al diseño proporcionado) y en la interactividad lógica mediante JavaScript.
1. **Inicio** 
- Presenta la identidad institucional con la misión, visión y valores de INTECAP.
- Incluye la integración de Google Maps para mostrar la ubicación del centro y enlaces dinámicos a redes sociales.
2. **Gestión de Empleados:**

- Un módulo interactivo para el registro de personal.

- Permite agregar nuevos registros a una tabla dinámica y ofrece funciones para eliminar el primer o último registro de la lista.

3. **Cálculo de Sueldo Líquido:**

- Calculadora financiera que procesa ingresos (salario, bonificación, comisiones) y egresos (ahorro, préstamos).

- Aplica automáticamente la retención del IGSS (4.83%) sobre el salario base para determinar el total neto a recibir.

4. Módulo de Indemnización:

- Herramienta avanzada para calcular la compensación económica de un empleado al finalizar su relación laboral.

- Utiliza fórmulas legales para determinar montos proporcionales de Bono 14 y Aguinaldo, sumando salarios pendientes y restando deudas para obtener el total final.
